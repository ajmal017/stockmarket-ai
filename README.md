## Run It Locally
```
git clone https://github.com/dopevog/stockmarket-ai.git
cd stockmarket-ai
pip3 install smtplib, pandas, pandas_ta, pandas_datareader, datetime, yfinance, colorama, rich
python3 main.py
```
